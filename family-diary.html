需要配置 Firebase
应用需要连接 Firebase 才能运行。
按以下步骤配置，只需一次。
第 1 步
进入 Firebase 控制台
打开 console.firebase.google.com，创建新项目（或使用孕期工具的项目）
第 2 步
启用 Realtime Database 和 Storage
在左侧菜单依次开启 Realtime Database（选亚洲节点）和 Storage，规则均设为测试模式
第 3 步
获取配置并填入文件
项目设置 → 你的应用 → 复制 firebaseConfig，替换本文件底部 <script> 里的占位内容：
const FIREBASE_CONFIG = {
  apiKey: "YOUR_API_KEY",
  databaseURL: "...",
  storageBucket: "...",
  ...
};
